stack · task + motion planning oct – dec 2025

Task and motion planning for robotic manipulation

A complete TAMP pipeline fusing symbolic task planning (PDDL) with sampling-based motion planning (OMPL) for autonomous block manipulation in the Genesis simulator — with an execution-replanning loop that recovers from object slipping and tower collapse.

Built a hybrid AI system combining logical reasoning over PDDL predicates with continuous geometric planning for collision-free trajectories. The symbolic layer uses Pyperplan to decompose goals into action primitives; the motion layer uses OMPL sampling-based planners to realize each action in the workspace.

Benchmarked seven planners — RRT, RRT*, RRTConnect, PRM, EST, FMT, and BIT* — evaluating trade-offs between planning time, path optimality, and success rate across varying initial configurations.

The execution-replanning loop handles real-world failure modes: when blocks slip or towers collapse mid-task, the system automatically re-grounds symbolic predicates from perception and re-plans dynamically, enabling robust multi-block stacking under physical uncertainty.

02 / 05
stack · state estimation apr – may 2025

Stereo visual-inertial odometry (S-MSCKF)

A stereo VIO system combining physics-based IMU propagation with learned feature detection and robust outlier rejection — a reusable EKF framework delivering real-time 6-DoF pose estimation on the EuRoC benchmark.

Implemented stereo VIO fusing IMU dynamics with visual feature tracks through an Extended Kalman Filter with sliding-window poses and inverse-depth feature triangulation. The filter design emphasizes robust outlier rejection, which is critical for stable convergence during aggressive motion or when visual features are intermittent.

The architecture combines model-based state propagation (physics of rigid-body motion from IMU) with data-driven front-end (feature detection) — a hybrid filter-based approach that outperforms purely learned VIO on challenging EuRoC MAV sequences.

Delivers real-time 6-DoF pose required for autonomous drone navigation and SLAM without GPS, with demonstrated generalization across multiple MAV sequences in the dataset.

stack · 3d scene reconstruction mar 2025

Buildings built in minutes — SfM + NeRF reconstruction

A dual-pipeline 3D reconstruction system: classical Structure-from-Motion for precise pose estimation paired with Neural Radiance Fields for dense photorealistic view synthesis from sparse image inputs.

Reconstructed photorealistic 3D models of large-scale architecture from only 2D image sequences, navigating the trade-offs between geometric accuracy (SfM's strength) and photorealistic rendering (NeRF's strength) for complex urban scenarios.

Engineered the classical front-end with COLMAP for SfM pose recovery, then the neural back-end with coordinate-based MLPs for NeRF volumetric rendering. Integrated through PyTorch.

Outperformed baselines on high-frequency details and complex non-Lambertian lighting — shiny surfaces, reflections, and specular materials — which break simpler reconstruction methods.

05 / 05
stack · classical cv foundations jan – feb 2025

MyAutoPano — deep homography vs. classical stitching

A seamless panoramic image-stitching system with two alternative pipelines: a classical Shi-Tomasi + RANSAC solver versus a VGG-style CNN trained with photometric loss. Deep learning approach reduced end-point error by 32%.

Built a panoramic stitching system for multi-view images with significant overlap, aiming to beat the computational intensity and brittleness of traditional feature-based alignment methods on real-world imagery.

Designed and benchmarked two distinct pipelines: a classical solver using Shi-Tomasi corners with RANSAC homography, and a deep-learning VGG-style CNN trained with photometric loss for direct homography estimation. Evaluated on MSCOCO and custom datasets.

The deep approach demonstrated superior robustness in texture-less regions and under large-baseline displacement, while the classical approach remained competitive on well-textured scenes — a useful lesson in when not to reach for deep learning.
